# 🎬 Netflix Login Clone

## 📌 Project Overview

Netflix Login Clone is a full-stack web application developed using **React.js**, **Node.js**, and **Express.js** that replicates the login experience of Netflix. The project focuses on implementing a complete authentication flow, including frontend validation, backend API communication, credential verification, and user redirection.

The application features a clean and responsive user interface inspired by Netflix's minimalist design. Users can enter their email and password, which are validated on the frontend before being sent to the backend. The backend performs mock authentication using static credentials and returns appropriate success or error responses.

## 🚀 Features

* Netflix-inspired login page design
* Responsive user interface for desktop and mobile devices
* Form handling using React state management
* Frontend validation for empty input fields
* REST API integration using Axios
* Express.js backend for handling authentication requests
* Mock credential validation without a database
* Error handling for invalid login attempts
* Successful login redirection to a Dashboard page
* Clean and modern UI built with Tailwind CSS
